# Part 1 - Data Cleaning

## Overview

This assignment focuses on cleaning and preparing the provided orders dataset for analysis. The dataset was reviewed for data quality issues, cleaned using Excel, and summarized using pivot tables.

## Work Completed

The following tasks were performed:

1. Reviewed the raw dataset.
2. Identified duplicate and inconsistent records.
3. Cleaned and standardized the data.
4. Applied business rules and validation checks.
5. Created a data quality report.
6. Generated pivot table summaries for analysis.
7. Documented the cleaning process in a cleaning log.

## Pivot Reports Created

* Sales and Profit by Region
* Sales and Profit by Category
* Order Count by Ship Mode
* Profit Margin by Segment
* Refunded/Cancelled/Failed Orders by Region
* Monthly Sales Trend
